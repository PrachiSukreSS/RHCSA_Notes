     there are 7 fields in /etc/passwd file:
     1. username, passwd mask, userid, groupid, GECOS name, home directory of user, shell access of user
   #modification of userdata in RHEL
   > usermod [options][arguments] [username]
   eg: usermod -s /sbin/nologin nisha
   options:(to change above mentioned fields)
   -u = uid
   -g = gid
   -c = GECOS name //ful name 
   -d = home directory change
   -s = shell
   -l = login name //change username
   eg:
  1. usermod -u 111 prachi
   check : cat /etc/passwd
  2. usermod -g 100 prachi
   check same
  3. usermod -c manager prachi
   check
  4. usermod -d /mnt prachi (usually do not change this)
  5. usemod -s /sbin/nologin prachi
  6. usermod -l xyz prachi
  
  
  or useradd -u 222 ram (by default username ram)

## Group management 
group  - collection of users
location - /etc/group
fiels - 1. grp name
      2. password mask 
      3. group id
      4. list of members 

  primary - every user 1 permission group -> -g
  secondary - user can be part of multiple  group -> -G

  >groupadd groupname
  check: cat /etc/group

  //add user
  > userass
  
  //assign myuser to linux group
  >groupadd -G linux myuser
  >cat /etc/group
  >id myuser

 //add user to mygroup
 >>groupadd -G mygroup myuser (not in linux anymore)

 //to add in multiple grps
 >usermod -aG myuser
